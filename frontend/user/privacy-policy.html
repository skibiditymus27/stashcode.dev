Privacy Policy

Last updated: March 2026

1. Introduction

StashCODE is a local-first, open-source Rust IDE. We believe your code is yours. This policy explains the minimal data practices behind the project and its website.

2. What We Collect

The StashCODE IDE itself collects no telemetry, analytics, or usage data by default. Your source code, workspace configuration, terminal sessions, and plugin state remain entirely on your local machine.

When you visit this website, standard server logs (IP address, browser user-agent, timestamps) may be recorded by the hosting infrastructure for security and uptime monitoring. No cookies or third-party trackers are used.

3. Plugin Sandboxing

Third-party WASM plugins run inside a capability-based sandbox. A plugin cannot access your filesystem, network, or clipboard unless you explicitly grant that permission at install time. Permission grants can be revoked at any point.

4. External Services

Certain optional features connect to external servers:

  • LSP servers — language intelligence runs locally by default (e.g., rust-analyzer). If you configure a remote LSP, data is governed by that server’s privacy policy.
  • Git remotes — push/pull operations contact your configured remote (GitHub, GitLab, etc.).
  • Plugin registry (planned for v1) — browsing the marketplace will send search queries to our registry API. No account is required.

5. Data Retention

Server access logs are retained for a maximum of 30 days and are not shared with third parties. We do not sell, rent, or trade any data.

6. Your Rights

Because we store virtually no personal data, there is usually nothing to delete. If you believe we hold data about you and want it removed, open an issue on our GitHub repository and we will respond within 14 days.

7. Changes

This policy may be updated as the project evolves (e.g., when the plugin marketplace launches in v1). Material changes will be announced in the IDE release notes and on this page.

8. Contact

Questions? Open an issue at github.com/skibiditymus27/stashcode.dev.
