Why StashCODE?

Built from day one around the principles that matter.

⚡

Blazing Fast

Fully async architecture powered by Tokio. Non-blocking LSP, terminal, and Git services keep the UI thread free at all times.

🔒

Sandboxed Plugins

WASM-first plugin SDK with capability-based permissions. Third-party extensions can never access your filesystem without explicit consent.

🐧

Linux Native

Designed for Linux desktops from scratch. Native Wayland support via Iced/wgpu, tight integration with system toolchains, and first-class terminal emulation.

🛠

Rust All the Way

Memory safety without garbage collection. The entire editor core, buffer model, and service layer are written in safe Rust with zero unsafe blocks in user-facing code.

🔌

Smart LSP

Deep Language Server Protocol integration with rust-analyzer, providing real-time diagnostics, completions, and inline type hints out of the box.

🏗

Open Architecture

Modular crate-based workspace. Swap out the UI layer, extend services, or build your own CLI tooling on top of the same core engine.

Workspace Architecture

Five crates, one coherent system.

core/
Editor buffer, rope data structure, and language model integration. The beating heart of every keystroke.
ui/
Wireframe shell built on Iced with wgpu rendering. GPU-accelerated text layout and theming engine.
services/
Async service layer: LSP client, PTY terminal multiplexer, Git operations, and workspace file indexer.
plugins/
WASM-first plugin SDK, host runtime with capability sandboxing, and sample extension for reference.
cli/
Runnable CLI demos that exercise the core and services without the full UI — perfect for CI and scripting.
